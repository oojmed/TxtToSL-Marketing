TxtToSL

Translate text into sign languages, quick and easy.

What?

TxtToSL's services translate text into sign languages. We currently support 8 sign languages in our API, and 6 sign languages in our web app. However, they have varying support, as can be seen in the sign language dropdown in TxtToSL Web.

We offer several services:


TxtToSL Web

Our web app for using TxtToSL API. It is (free and) open source. You can find the app here


TxtToSL API

Our API for everything, it is free, but not open source. You can see the docs here.

Why?

"I wanted to make a Google Translate-like translater. Not always 100% accurate but still alright, as well as easy and simple to use."

- Oj, Director of Oojmed


When researching for other similar applications (text to sign language translators), we found that none of them:

  • Use videos of real people signing
  • Have free, public APIs
  • Are as flexible, with several sign languages avaliable in many languages, and hosting options

Development is led by Oojmed, a non-profit software development organisation. All of our code (including this site) except the API / server is open source on our GitHub profile.
